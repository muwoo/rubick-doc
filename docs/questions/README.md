# 常见问题
## rubick 点击开启网络抓包无反应
::: tip
command + shift + L 打开控制台，查看报错信息后进行反馈，
一般都是权限问题，可以使用 
` sudo /Applications/rubick.app/Contents/MacOS/rubick` 
来使用管理员权限打开rubick
:::

:::
